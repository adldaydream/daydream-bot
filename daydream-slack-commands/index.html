Daydream Slack Bot

Available Commands

/daydream-count
Get the current count of submissions. Returns a fun, random celebratory message with the count.
/responses
Link to view daydream responses.
View Responses
/github
Link to the Daydream Adelaide GitHub repository.
GitHub Repo
/days-until
Countdown to September 27th. Tells you how many days are left, with special messages for the day before and the day of!
🔐 Bot Permissions
  • Read messages and slash commands
  • Post messages in channels where invited
  • Access to Fillout API for submission counts
  • No access to private messages or DMs
  • Cannot read message history before installation
